# EWPMS
# Executive Water Projects Management System

منصة متكاملة لإدارة ومتابعة مشروعات مياه الشرب والصرف الصحي، تم تطويرها خصيصًا لدعم الجهاز التنفيذي لمشروعات مياه الشرب والصرف الصحي بمحافظة الفيوم، مع إمكانية التوسع لتخدم جميع فروع الهيئة القومية.

---

## أهداف النظام

- متابعة جميع المشروعات من بداية التنفيذ حتى التسليم النهائي.
- متابعة البرنامج الزمني والانحرافات.
- متابعة الشركات المنفذة والاستشاريين.
- متابعة القرارات حتى الإغلاق.
- إدارة الخطابات والمكاتبات.
- إصدار تقارير تنفيذية للإدارة العليا.
- توفير Dashboard لحظي لرئيس الجهاز.
- دعم اتخاذ القرار باستخدام مؤشرات الأداء.
